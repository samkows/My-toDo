# ToDo App (Java → Kotlin) 

<details>

## 📱 О проекте
Приложение для управления задачами с:
- Миграцией с Java на Kotlin  
- Архитектурой MVVM  
- Room для хранения данных  

## 🛠 Технологии  
- **Язык**: Kotlin (было Java)  
- **Архитектура**: MVVM + Clean Architecture  
- **База данных**: Room  
- **Асинхронность**: Coroutines
